## Introduction

This tutorial guides you through setting up a **fully functional Google Image Agent** using **Agentica**. With the following code block, you can create an agent that interacts with Google Image, powered by **OpenAI's GPT model**.

## The Complete Google Image Agent

First, install the required Google Image connector package:

```bash
npm install @wrtnlabs/connector-google-image
```

Then, add the following code to your project:

```typescript
import { Agentica } from "@agentica/core";
import { GoogleImageService } from "@wrtnlabs/connector-google-image";
import dotenv from "dotenv";
import OpenAI from "openai";
import typia from "typia";

dotenv.config();

const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

export const GoogleImageAgent = new Agentica({
  model: "chatgpt",
  vendor: {
    api: openai,
    model: "gpt-4o-mini",
  },
  controllers: [
    {
      name: "Google Image Connector",
      protocol: "class",
      application: typia.llm.application<GoogleImageService, "chatgpt">(),
      execute: new GoogleImageService({
        apiKey: process.env.SERP_API_KEY!,
      }),
    },
  ],
});
```

## What This Does

```env
OPENAI_API_KEY=your-openai-api-key
SERP_API_KEY=your-serp-api-key
```
